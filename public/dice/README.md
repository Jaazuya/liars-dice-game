# Imágenes de Dados

Coloca aquí las imágenes de los dados (1-6).

## Nombres de archivos requeridos:
- `dice-1.png` - Dado con 1 punto
- `dice-2.png` - Dado con 2 puntos
- `dice-3.png` - Dado con 3 puntos
- `dice-4.png` - Dado con 4 puntos
- `dice-5.png` - Dado con 5 puntos
- `dice-6.png` - Dado con 6 puntos

## Recursos para descargar imágenes:

1. **OpenGameArt** (https://opengameart.org)
   - Busca "dice" o "d6"
   - Muchas opciones gratuitas

2. **Kenney.nl** (https://kenney.nl/assets)
   - Assets de juegos gratuitos
   - Tiene sets de dados

3. **Itch.io** (https://itch.io/game-assets/free)
   - Assets gratuitos de la comunidad

4. **Crear tus propias imágenes SVG:**
   - Puedes crear dados simples con SVG
   - O usar herramientas como Figma, Canva, etc.

## Nota:
Si no agregas las imágenes, el componente usará automáticamente los emojis de dados como fallback.

## Tamaño recomendado:
- 128x128 píxeles o más
- Formato PNG con transparencia
- Fondo transparente
